# Compile and Install #

cd totem-plugin-timer

./configure

make

sudo make install


# Enable plugin in totem #

Under 'Edit' pull-down menu, select "Plugins".  In 'Configure Plugins' pop-up window, click button at left of plugin labelled 'Timer'.

# Using plugin #
